Leedcode 67: Add Binary

Given two string a and b, return their sum as a binary string.

Input: a = '101', b = '111'
Output: '1100'

Idea: 1. ans can use list and string as the result in a string. if use the string: + if use the list need to add the "".join()[::-1] at the end

  1. while loop instead the for loop: the length is unsure, both use the index

  2. var: res, carry, initialize the value.

  3. the last value in both list is not add in as the carry is replaced.

class Solution:
    def addBinary(self, a: str, b: str) :
        ans, extra = [],0 
        i,j=len(a)-1,len(b)-1
        while i>=0 or j>=0:
            if i >= 0:
                extra += int(a[i])
            if j >= 0:
                extra += int(b[j])
            ans.append(extra % 2)
            extra //= 2
            i,j = i-1,j-1
        if extra == 1:
            ans.append(1)
        ans = [str(x) for x in ans][::-1]
        return ''.join(ans)
